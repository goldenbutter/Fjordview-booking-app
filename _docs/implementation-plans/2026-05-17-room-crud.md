# Room CRUD Implementation Plan

> **For agentic workers:** S07 is executing this plan inline with TDD. Keep steps checked as they land.

**Goal:** Turn `/admin/rooms` from read-only inventory into scoped admin management for room types and physical rooms.

**Architecture:** Keep `/admin/rooms` as one operational page. Server components load the current property snapshot; small client forms call admin API route handlers. Route handlers resolve `getCurrentAdminContext()` and pass only that property id into DB helpers.

**Tech Stack:** Next.js App Router, React client components for forms, Drizzle query helpers, Node `tsx --test` static/behavior tests, existing Tailwind/UI primitives.
